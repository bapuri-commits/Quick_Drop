## 핵심 결정 & 맥락
- **프레임워크 없이 단일 HTML**: 번들링 불필요, nginx 정적 서빙 용이, 빠른 개발
- **CLI 표준 라이브러리만 사용**: pip install 없이 어디서든 `python quickdrop.py`로 즉시 실행
- **파일 메타데이터를 JSON 파일로 관리**: DB 없이 심플하게. 개인용이라 파일 수가 적어서 충분
- **세션 쿠키 인증**: 웹에서는 쿠키, CLI에서는 토큰 직접 전달. 30일 유효
- **포트 8200**: 다른 서비스(The_Agent 8000, News_Agent API 8100)와 충돌 방지
