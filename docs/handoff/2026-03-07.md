# QuickDrop Handoff — 2026-03-07

## 현재 상태
VPS 라이브 배포 완료 (`http://46.250.251.82:8200`). 데스크탑/노트북/핸드폰 모두 정상 동작 확인. systemd 서비스로 상시 실행 중.

## 작업 내용

### Phase 1 — MVP (백엔드 + CLI)
- FastAPI 백엔드: drop(임시 전송, 자동 만료) + vault(영구 보관, 폴더 구조)
- CLI 클라이언트: 표준 라이브러리만 (login/upload/download/list/delete + vault)
- Docker 설정 + cleanup.py (cron용)

### Phase 2 — 웹 UI
- 임시 전송 / 보관함 탭 분리
- vault 폴더 탐색기 (breadcrumb, 새 폴더, 업로드, 삭제)
- 만료 셀렉터를 드롭존 내부로 이동 (UX)
- 파일 검색 (drop/vault 각각)
- 이미지/비디오 썸네일 미리보기
- 새로고침 버튼

### 모바일 대응
- `<label for>` 방식으로 파일 선택 (모바일 터치 호환)
- `accept="*/*"` 속성 추가
- 썸네일 인증: 토큰 3단계 폴백 (window._qdToken → localStorage → cookie)
- 쿠키 `SameSite=lax`, `httponly=false`로 변경

### 엣지케이스 처리
- 연결 끊김 → "연결 실패 — 네트워크를 확인하세요"
- 업로드 타임아웃 (10분) → "시간 초과"
- 인증 만료 중 업로드 → 로그인 화면 자동 전환
- 서버 에러 → HTTP 상태 코드 표시

### VPS 배포 (DevOps Stage 2)
- SSH 키: 데스크탑 `C:\Users\user\.ssh\id_ed25519`, 노트북 `C:\Users\chois\.ssh\id_ed25519`
- `/opt/apps/quickdrop/` (코드) + `/opt/data/quickdrop/` (데이터) + `/opt/envs/quickdrop.env` (시크릿)
- systemd `quickdrop.service` enabled + active
- ufw 8200/tcp 개방

## 핵심 결정 & 맥락
- **이중 저장 구조**: drop(임시, 만료) + vault(영구, 폴더) 분리
- **CLI 표준 라이브러리만 사용**: 어디서든 `python quickdrop.py`로 즉시 실행
- **토큰 인증 이중화**: 쿠키 + URL 쿼리 파라미터 (`?token=xxx`) 병행으로 모바일 호환
- **포트 8200**: The_Agent(8000), News_Agent(8100)과 충돌 방지
- **VPS 비밀번호 아직 기본값**: changeme → 변경 필요
