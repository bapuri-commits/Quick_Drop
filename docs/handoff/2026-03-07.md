### vault 추가 (Phase 1 후반)
- `backend/config.py`: `VAULT_DIR`, `MAX_VAULT_BYTES` 추가
- `backend/main.py`: vault API 5개 엔드포인트
  - `GET /api/vault?path=` — 폴더 목록 (파일/하위폴더, 크기, 수정일)
  - `POST /api/vault` — 파일 업로드 (path 파라미터로 폴더 지정)
  - `GET /api/vault/download?path=` — 파일 다운로드
  - `DELETE /api/vault?path=` — 파일/폴더 삭제 (폴더는 재귀 삭제)
  - `POST /api/vault/mkdir` — 폴더 생성
- `cli/quickdrop.py`: `vault` 서브커맨드 (list, upload, download, delete, mkdir)
- 디렉토리 탐색 보안: `_safe_vault_path()`로 path traversal 방지

## 핵심 결정 & 맥락
- **이중 저장 구조**: drop(임시, 자동 만료) + vault(영구, 폴더 구조) 분리
- **vault는 원본 파일명/폴더 구조 그대로 저장**: 메타데이터 JSON 불필요, 파일시스템이 곧 구조
- **CLI 표준 라이브러리만 사용**: pip install 없이 `python quickdrop.py`로 즉시 실행
- **포트 8200**: The_Agent(8000), News_Agent API(8100)과 충돌 방지
- **Windows cp949 이슈**: CLI에서 이모지 대신 `[D]`/`[F]` 텍스트 사용
- **UTF-8 BOM 이슈**: PowerShell이 BOM 포함 JSON 생성 → `utf-8-sig` 디코딩으로 해결
